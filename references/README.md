# References

External feedback, literature summaries, and reviewer comments. Synthesized notes only -- not raw PDFs or data files.

## What Goes Here

- AI feedback on your approach (e.g., Claude/ChatGPT/Perplexity reviews of your experiment design)
- Literature review summaries (key findings from relevant papers)
- Reviewer comments (from collaborators, lab meetings, peer review)
- Conference/workshop feedback

## Naming Convention

`[source]_[topic]_[date].md`

Examples:
- `claude_architecture_feedback_2026-04-10.md`
- `literature_review_data_augmentation_2026-04-15.md`
- `lab_meeting_feedback_2026-05-01.md`

## How to Use

Summarize key points here. Reference these files from `docs/critical_assessment.md` (External Feedback section) when incorporating outside perspectives into your project assessment.
